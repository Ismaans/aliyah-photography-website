# How to Share the Website & Add Photos Easily

## Option 1: Share via Cloud Storage (Easiest for Quick Sharing)

### Step 1: Upload to Google Drive / Dropbox / iCloud
1. Zip the entire website folder
2. Upload to Google Drive, Dropbox, or iCloud
3. Share the link with your cousin
4. She downloads and opens `index.html` in a browser

**Pros:** Quick and simple  
**Cons:** She needs to manually update files and re-upload

---

## Option 2: Deploy to GitHub Pages (Free & Professional)

### For You (Initial Setup):
1. Create a GitHub account at github.com (if you don't have one)
2. Create a new repository (make it public)
3. Upload all website files
4. Go to Settings → Pages → Select "main" branch → Save
5. Your site will be live at: `https://yourusername.github.io/repository-name`

### For Your Cousin (Adding Photos):
1. She creates a GitHub account
2. You add her as a collaborator to the repository
3. She can edit files directly on GitHub:
   - Click on `work.html` or `script.js`
   - Click "Edit" (pencil icon)
   - Add new photos to the `images/` folder
   - Update the HTML/JavaScript with new image paths
   - Click "Commit changes"
   - Changes go live automatically!

**Pros:** Free, professional, automatic updates  
**Cons:** Requires basic GitHub knowledge

## Quick Guide: How to Add Photos (For Your Cousin)

### Method 1: Simple File Replacement
1. Add new photos to the `images/` folder
2. Name them: `work-7.jpg`, `work-8.jpg`, etc.
3. Open `work.html` in a text editor
4. Copy an existing work item and paste it
5. Change the image path to the new photo
6. Update the title and project name

### Method 2: Using the JSON File (Easier)
1. Open `photos.json` in a text editor
2. Add a new project entry following the same format
3. Save the file
4. The website will automatically show the new photos
